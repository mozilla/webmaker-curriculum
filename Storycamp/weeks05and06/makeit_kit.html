Teaching Kit

Make it! Share it! Teaching Kit

These activities will help people learn how to plan and make an interactive and remixable How-to that uses the Web as it's platform. See all Storycamp Resources

Made by Laura Hilliger

What is this kit all about?

The Make it! Ship it! Kit will guide learners in planning, prototyping and producing How-to videos that pull in dynamic data from the web.

Start with these videos, which contain valuable information on how to think about and plan Web Native Films.

Learning objectives

Learners will understand:

  • Collaboration makes ideation more fun and successful.
  • Information architecture is a system of organization that takes into account both how you present information and how a user accesses that information.
  • SVT (Story, Vision, Tech) is a solid model for creating a web native film. Technology can serve as inspiration, but never substitute for a story.
  • Using data allows you to augment your video with extra information.

Learners will be able to :

  • Plan a How to tutorial using paper prototyping.
  • Create the How to using Popcorn Maker
  • Augment the tutorial by adding data to it.
  • Use keyboard shortcuts (i.e. for copying and pasting)
  • Right-click and scroll with the mouse
  • Create storyboards and schematics.

What you'll make together

Learners will use Popcorn Maker to make their own how to tutorials using dynamic data and the web as their platform. They will collaborate to make learning resources and use assets from the web to explain something they would like others to be able to do.

Preparation

Make sure computers have the latest version of Firefox or Chrome on them. Help learners set up a Persona account before the session, so that they can save their work.

Discussion Questions

  • What does information architecture mean?
  • What does "open web" mean?
  • How did feedback help you iterate and improve your project?
  • Why is it important to have open access to the Internet?
  • Why is it important that the web is designed by many individuals?
  • Why is it important for websites to invite participation?
  • Why is it important to comment on other people's work, ideas or commentary?

Assessment criteria

Did the learner show their ability to:
  • add information to media to create something original
  • understand and use Popcorn
  • find content to remix and reuse
  • change the story of a piece of media by adding information to it
  • use a search engine to find specific content
  • find the content he/she was looking for
  • use creativity
  • work with others
