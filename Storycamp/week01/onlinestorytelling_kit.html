Teaching Kit

Online Storytelling

The Online Storytelling activities delve into Digital Storytelling using Popcorn Maker as a mechanism for creative expression through story on the Web. See all Storycamp Resources

Made by Laura Hilliger

What is this kit all about?

Procedural storytelling refers to a way of designing stories so they’re different every time, and the open structure of the Web allows you to create stories using this principle. It will also introduce concepts like information architecture and web native films by helping the learner change the context of a story by using dynamic data in video.

Learning objectives

Learners will understand:

  • Information architecture is a system of organization that takes into account both how you present information and how a user accesses that information.
  • SVT (Story, Vision, Tech) is a solid model for creating a web native film and other web native stories. Technology can serve as inspiration, but never substitute for a story.
  • Procedural storytelling refers to a way of designing stories so they’re different every time. It is just one of the unique aspects of Web Native Film.
  • the open structure of the Web allows you to create procedural stories
Learners will be able to :
  • Use Popcorn Maker to pull dynamic data from the web into a video.
  • Use keyboard shortcuts (i.e. for copying and pasting)
  • Right-click and scroll with the mouse

What you'll make together

Learners will use Popcorn Maker to make their own web native films and explore larger concepts on the topic. They will collaborate to make critical commentary and use assets from the web to tell a story of their own.

Preparation

Make sure computers have the latest version of Firefox or Chrome on them. Help learners set up a Persona account before the session, so that they can save their work.

Discussion Questions

  • What does information architecture mean?
  • What does "open web" mean?
  • What is procedural storytelling?
  • How did feedback help you iterate and improve your project?

Assessment criteria

Did the learner show their ability to:
  • be creative in finding a story
  • voice opinions about interactivity
  • explain procedural storytelling
  • bring data into a film
  • change the story of a piece of media by adding information to it
  • use a search engine to find specific content
  • find the content he/she was looking for
  • use creativity
  • work with others
