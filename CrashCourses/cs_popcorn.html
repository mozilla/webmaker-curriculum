Cheatsheet

Popcorn Maker

What this page is

The Popcorn Maker Cheatsheet is a quick reference guide to understand Popcorn Maker and for the keyboard shortcuts Popcorn Maker uses.

Popcorn's Basic Features

  1. Projects

    A project provides a visual layout and is usually designed for a specific kind of content. Clicking the +Events Button at the top right of the interface, allows you to see all the plug-ins you can use in your project. You have to drag the plug-in you’d like to use and drop it onto the canvas or into the timeline to add it to your project.

    You will see the project gallery at popcorn.webmaker.org. You can also create a project from scratch.

  2. Events

    An event is a marker that says “do something” at a specific point in your film. To add an event to the timeline, you drag and drop the text from the Events box to the Timeline! To delete an event from the Timeline, you click the event and hit the Delete key on your keyboard.

    Popcorn has a great many plug-ins which allow you to pull events into your timeline. Because there are many plug-ins and projects to remix, there are infinite variations and possibilities. No two Popcorn projects will be the same. But the authoring pattern is always the same: choose a plug-in, add it to the timeline, and configure the event.

  3. Layers

    A Popcorn Maker project can have an infinite number of layers. Layers help you, the creator, organize events in the way that seems most logical to you. Maybe you want one layer to visualize all the Twitter events. Or maybe you want one layer to visualize all the cat-related events. It’s up to you. When you mouse over the name of a layer, you’ll see a trashcan to the right. This allows you to delete the layer.

  4. Saving, Exporting and Publishing

    You can save your project, export the code of your project and/or publish your project to the WWW. To save, you'll need to create a Persona account. Just follow the on screen instructions and share your masterpieces with the world!

Keyboard Shortcuts

Spacebar allows you to Play and Pause the video playback. Using this shortcut will also update the timeline view if the playhead gets lost.

If you click an event and hold down SHIFT, then click another event, you can move both events on the timeline in sync using the arrow keys.

Using the up/down arrow keys will allow you to move an event to a different track.

Click on an event and hit the delete key to remove it from the timeline.
