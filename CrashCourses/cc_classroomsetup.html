How to Begin with the Goggles

Tips for Classroom Preparation

What this page is

This page will help you prepare your classroom and gather resources to use the TASCasaurus Kit and the XRay Googles Kit.

Set up your room

  1. Set up a welcome table in the front of the room

    The table should include helpful handouts; sign in sheets; guide sheets; posters; a workshop schedule; and other materials you feel are necessary. See the sidebar to the right for some helpful resources!

  2. Set-up participant computers
    • Make sure you have enough computers available for participants or teams of participants.
    • Test the computers to make sure they work.
    • Make sure all computers are fully charged and/or have a power source.
    • Make sure the room has a stable Internet connection that can support the number of participants expected.
    • If necessary, install the most up-to-date version of Firefox, or Safari on every computer (please note that Hackasaurus DOES NOT work on Internet Explorer).
    • Pre-install theX-Ray Goggles. If the Goggles do not work, it is probably because the browser you are using is outdated.
  3. Set-up mentor's computer
    • Install X-Ray Goggles and open all necessary webpages/files for the lesson.
    • Connect instructor computer to a projector or smart board for teacher intro and demonstration (if available). Note: YOU DO NOT have to have a projector/smartboard for the lesson, but it can be helpful.
  4. Set up a flickr account for uploading pictures
    • Ideally, your youth will be able to scan/download pictures onto the computers as they create them.
    • If not, create and download in advance pictures of the objects for the youth to upload to flickr. You should have at least one picture for each person/team to upload for practice.
    • Note: pictures must be in .jpg or .png format. If you scan drawings, make sure to save them in one of those formats.
    • Please note: You should discuss with your supervisor how to provide access to flickr. You may want to open one account that all the youth can use, e.g." PS zzz afterschool," rather than having each person open her own. Also make sure to choose the privacy settings you want. The uploaded pictures can be available publicly or kept private.

Guides and Resources

  • Photocopy guide sheets

    It's helpful to have guide sheets printed and available for participants to follow. The following are helpful guides for learners:

    • HTML Examples
    • HTML Crash Course
    • HTML Cheatsheet
    • CSS Cheatsheet
    • XRay Goggles Cheatsheet
    • Decode the Code Challenge

  • Identify and have available resources

    You might want to have the resource links listed on paper and/or display them on a wall, chalkboard or smartboard. Here are a few links that may help with learning code:

    • http://webmaker.org/
    • https://developer.mozilla.org/en-US/learn
    • http://www.w3schools.com/
    • http://www.tizag.com/

    Provide list of STEM websites for students to begin hacking and for research. See Biodiversity Prep for suggested sites, or use others familiar to you.
