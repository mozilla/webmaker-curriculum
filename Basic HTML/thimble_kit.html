Teaching Kit

Jumping into HTML with Thimble

Help learners get the basics of HTML by making their first web page. Use Thimble to create new pages and instantly see how changes in code will affect the rendered page. Then publish and share the page, which others can later remix.

Made by: Mozteach

What is this kit all about?

The Web is made out of building blocks that anyone can rearrange to create new things, and HTML is a language used to tell computers how to build webpages so that humans can consume them. With this kit, learners will be introduced to HTML and CSS while they build their first webpages. They'll also verify sources on the web and explore what it means to have an online identity.

Learning objectives

Learners will understand:

  • The combination of an opening tag and its corresponding closing tag and the content in between is called an element
  • Parts of a webpage include HTML, CSS, graphic assets and text content
  • CSS means cascading style sheets
  • It is important to write code following standards so that it is accessible to different kinds of users
  • There are resources on the web for webmaking
  • What information is public on the web
  • How to verify sources on the web
  • What Mozilla means when we say “hack” & “webmaking”

Learners will be able to :

  • Create their own webpages using the following HTML tags: <html>, <head>, <title>, <body>, <h1>,<h2>, etc, <p>, <ul> & <ol>, <li>, <img>, <a>
  • Style their webpages using the following CSS properties: positioning values, hex numbers/color, fonts/sizes, class, div
  • Identify the address bar in a browser
  • Use multiple browser tabs
  • Select content on a webpage/address bar using “copy” and “paste”
  • Use the mouse
  • Be able to type non-alphanumeric characters like @, &, <, etc.
  • Identify and alter element tags
  • Remix content of element attributes
  • Remix and create elements
  • Demonstrate the ability to use HTML tags
  • Define the term CSS
  • Use CSS to style a page

What you'll make together

Learners will use Thimble to make several webpages.

Preparation

Make sure computers have the latest version of Firefox or Chrome on them. Help learners set up a Persona account before the session, so that they can save their work.

Discussion Questions

  • What is the web made out of?
  • What does "open web" mean?
  • How should you protect your data on the web?
  • How does HTML & CSS allow you to be creative?
  • How did feedback help you iterate and improve your project?

Assessment criteria

Did the learner show their ability to:
  • tweak or enter a URL
  • switch between browser tabs
  • right-click and scroll with the mouse
  • share a hacked page with peers
  • create content that remixes elements found on the Web
  • create and format text and images on a webpage using basic HTML tags
  • recognize HTML and CSS
  • use a search engine to find specific content
  • find the content he/she was looking for
  • revisit suggested websites for resources
  • use creativity
  • work with others
