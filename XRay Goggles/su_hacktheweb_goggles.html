Activity

Hack the Web with the Goggles

The goal of this activity is for participants to create their first webpage by hacking and tinkering with an existing webpage.

Made by Laura Hilliger

Steps for the Activity

  1. If you haven’t already, install the Goggles here. Tell participants that they will be hacking a website. Go to a website and turn on the Goggles. You can do this by clicking on the bookmarklet.

  2. Using the Goggles, slowly hover the cursor over objects so that participants can see that the tool is inspecting the code and showing the object tags for each object. Point out the tags, attributes and elements. Find the object tags that you want to change or replace (e.g. the p tag or img tag).

  3. Hit “R” on your keyboard or just click on an element. Show participants that the code is revealed. Ask participants for help hacking the code.

  4. After you have made the changes, hit the "Save Changes" button and show everyone the hack. Now, show everyone how to do this step by step. Then let learners play with the Goggles for a few minutes.

Materials

  • Projector
  • Computer
  • Web Access
  • HTML Cheatsheet
  • CSS Cheatsheet
  • Goggles Cheatsheet

Discussion

At the end of the session, have participants present their new webpages, explain their work process and what they learned.

Go over the steps again.

Look at the code, point out the tags, attributes and elements.

Hack one part of the webpage. Hit the “save changes” button to save changes locally on your computer.

Share: hit “P” to publish the new webpage on the Internet. Take a screenshot or copy and paste the code into a HTML document.
