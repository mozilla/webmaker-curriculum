Javascript Basics - aka Quacking Javascript

Learners get familiar with writing Javascript in this six-part module. Discover the foundations of the text coding through the production of simple web pages.

  • Preparation

    Review the activities below and decide which ones you'd like to use. Set up a discussion space your learners to share their learning and reflect on the web. Introduce yourself in the same discussion space, so we can support you better!

    Getting Started

    These activities are designed to be used in Computer Clubs, as guided sessison for learners of all ages or as a self directed online course. Please just skip any instructions that don't seem right for you.

    These introductory sessions are focusing on the use of variables, different data types and how they can be used to output simple statements to the page.

    1. What's in a Name

    1
    hr

    Introduction to Javascript Code and Variables
    A fun start to look at what variables are and why they are useful when we are creating webpages using Javascript.

    Try the activity on your own.
    Check out the facilitator's notes, and any tips in the forum for this course.
    Then do it with your learners.

    Magic with Numbers

    1
    hr

    Maths and Operators in JS
    Using the hidden parts of web browsers to find out more about numbers, maths and Data Types in Javascript.

    Try the activity on your own.
    Then do it with your learners.

    Simple Web Creations

    Here we start to learn some of the tools needed to create more interesting web pages.

    2. Cooking the Web

    1
    hr

    Cooking the Web
    To make web pages we need to learn some of the language of the net and then take steps one after the other carefully.

    Explore the activities by yourself.
    Then invite other learners to join you.

    Selection and Loops

    These activities take you into more advanced areas of coding. Here we start to thing about code that can save us time.

    5. Going Loopy
